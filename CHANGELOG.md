# CloudPlayer Changelog

使用方式：

- 日常开发时，把准备发版的内容持续补到 `## Unreleased` 下。
- 推 tag 触发 release 时，发布脚本会优先读取这个区块生成 release body。
- 发版完成后，运行 `./scripts/archive_unreleased_changelog.sh vX.Y.Z` 归档 `## Unreleased`，再继续在模板里累积下一版内容。

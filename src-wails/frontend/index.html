# 歌名 歌手 专辑

偏好设置

关闭主窗口行为、快捷键、网易云 API 根地址（可选）与桌面歌词颜色将保存到本地配置。播放页歌词固定为 QQ → 酷狗 → 网易云 → LRCLIB（与「换」歌词替换同源）。

界面模式

亮暗模式与主题色分开保存。选择「跟随系统」后，会随着 macOS 外观实时切换。

应用主题

预设共 6 种，再加 1 个自定义颜色模式。切换后主界面强调色与 macOS Dock 图标都会同步更新；菜单栏图标保持模板白色。

自托管 NeteaseCloudMusicApi 或兼容服务的根 URL（无尾斜杠亦可）。填写后「netease」源将优先请求 /lyric/new 获取 YRC 逐字歌词；留空则仅用网易云网页门户的行级 LRC。

后台也能响应；默认使用 Ctrl+Alt+ 组合以避免中文输入法占用 Ctrl+Space。点击快捷键框后按下新组合即可替换，Esc 取消、Backspace 清除；修改后自动生效。
